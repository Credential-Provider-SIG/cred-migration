                   The Credential Migration Protocol
                        draft-cpsig-crimp-latest

Abstract

   This specification defines a protocol to securely move one or more
   user credentials between two credential providing applications.

3.  Flow Diagram

   mermaid sequenceDiagram participant E as Credential Data Source
   participant I as Credential Data Destination note over E,I:
   Authorizing party I->>I: (1) Create key-pair for importer I->>E: (2)
   Send importer public key E->>E: (3) Create key-pair for exporter
   E->>E: (4) Create shared secret / symmetric key E->>E: (5) Encrypt
   local data with symmetric key E->>I: (6) Send exporter public key and
   encrypted bundle I->>I: (7) Re-create shared secret / symmetric key
   I->>I: (8) Decrypt bundle with symmetric key
